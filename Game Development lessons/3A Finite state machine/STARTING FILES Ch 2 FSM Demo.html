Book Example

Installing the starting files: Download and unzip this folder: Unity-AI-book-Chapter02-starting-files.zip. In Unity Hub choose "Add" > "Add project from disk" and select the unzipped folder.  If a warning says "To open this project, install the missing Editor version or open it with another version" click "Choose another Editor version" and choose your editor, NOT the original one. Each time it double-checks that you "really want to change editors," agree. It should open fine. 

Instructions are in the book at https://learning.oreilly.com/library/view/unity-artificial-intelligence/9781803238531/B17984_02_Epub.xhtml#_idParaDest-48. THE FILES ARE NEARLY DONE [A LOT MORE IS DONE THAN THE BOOK SAYS] Here's all you need to do:

  1. Open the Chapter 2 project
  2. From Assets > Scenes, open the scene “SimpleFSM.”
  3. With PlayerTank selected in the Hierarchy, notice in the Inspector the PlayerTankController script is attached to the PlayerTank. (It’s also in the Assets > Scripts > Misc folder.) Drag and drop Turret from the Hierarchy into the corresponding fields of the PlayerTankController script in the Inspector.
  4. Still looking in the Inspector at the PlayerTankController script, drag the Bullet Prefab from Assets > Resources > Prefabs into the script field for it.
  5. Selecting each enemy Tank one by one in the Hierarchy, look in the Inspector at the SimpleFSM attached to the Tank. Drag the Bullet Prefab from Assets > Resources > Prefabs into the script field for it.
  6. Select the Bullet Prefab from Assets > Resources > Prefabs and look at its Bullet Script in the Inspector. From Assets > Particle Effects, drag ParticleExplosion into the script field for it.
