Exiting a Fullscreen Game

To enable the player to quit your game while playing fullscreen, implement the line Application.Quit(); triggered by the user clicking an exit button or, as shown below, by the user hitting “esc”:

private void Update()

if (Input.GetKeyDown(KeyCode.Escape))

{

Application.Quit();

}

}
