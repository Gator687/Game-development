Since the starting files above do not include a project file, I made a new empty 3D project, then in my Assets folder I deleted the existing Scripts folder and dragged the contents of the downloaded Assets folder (but not the Assets folder itself) into my Assets folder in the Project view.

Instructions are in the book at https://learning.oreilly.com/library/view/unity-artificial-intelligence/9781803238531/B17984_04_Epub.xhtml. THE FILES ARE NEARLY DONE [A LOT MORE IS DONE THAN THE BOOK SAYS] Here's all you need to do:

  1. Open the Chapter 4 project
  2. From Assets > Scenes, open the scene “Demo.”
  3. With AI_Character (red enemy tank) selected in the Hierarchy, notice in the Inspector it has a Box Collider component (it fits snugly around the red tank). You have to check the checkbox marked "Is Trigger."
  4. With AI_Character (red enemy tank) selected in the Hierarchy, notice in the Inspector it has a Touch script that has a "Target Affiliation" property dropdown. Choose "Player" in that dropdown.
