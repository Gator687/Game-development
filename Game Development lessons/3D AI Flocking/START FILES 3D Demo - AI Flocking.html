Installing the starting files: Download and unzip this folder: Unity-AI-book-Chapter05-starting-files.zip. In Unity Hub choose "Add" > "Add project from disk" and select the unzipped folder.  If a warning says "To open this project, install the missing Editor version or open it with another version" click "Choose another Editor version" and choose your editor, NOT the original one. Each time it double-checks that you "really want to change editors," agree. It should open fine. 

Instructions are in the book at https://learning.oreilly.com/library/view/unity-artificial-intelligence/9781803238531/B17984_04_Epub.xhtml.

Here's what we do in the demo:

  1. Open the Chapter 5 project
  2. From Assets > Scenes, open the scene “FlockingUnity.” There are three UnityFlockControllers, each with 11 “UnityFlock” boids all occupying the same position to begin with. They’ll explode apart to get away from each other on start, then often plow through the ground.
  3. Hide UnityFlockControllers 2 and 3 and experiment... To get a tighter formation, in Inspector set for all 11 boids:
    1. minSpeed = 20
    2. toOriginRange = 17.0f;
    3. avoidanceRadius = 50.0f; //To lessen the initial “explosion”
  4. To get the flock to move in a straight line rather than randomly, in public class UnityFlock:

    void Start()

    {

    initialPosition = transform.position;

    //CalculateNextMovementPoint(); THEN ADD THIS NEXT LINE:

    nextMovementPoint = initialPosition + new Vector3(0, 0, -500);

    }

You can also open the scene named FlockingReynold. This one has the advantage that the boids generate using code, but they also don't seem to have the beautiful variety to their movement that they had in the other scene.

  1. Note there’s a FlockController again for the flock children...but no children.
  2. In the Project panel select the grey “Flock” cube Prefab (Assets > Resources > Prefabs) and note it has Rigidbody & Sphere Collider & Flock script. The Controller will generate these Flocks dynamically at start using Instantiate().
  3. In the Hierarchy there's a “Target” sphere entity as a moving leader for our flock to pursue. We've assigned its Transform in one of the scripts in the Inspector.
  4. Assign the Target sphere a bright color (Yellow Material from Assets > Resources > Prefabs folder, perhaps)
  5. Rotate Camera's transform position y=165 to face flock.
  6. Move the Target to x=0 y=10 z=0. 
  7. Move the First Person Controller to x=0 y=15 z=-22.
  8. Play the scene. Look to one side and down and see the yellow Target move in a circular motion. Look to another side and up and see the flock cubes fly in a circular motion. We can bring them closer to one another...
  9. Move the FlockController to x=0 y=15 z=0.
  10. Select the Target and find where its Target Movement script is connected in the Inspector. Set its Bound x=40 y=0 z=70.
  11. Play the scene and watch the boids follow just above the Target, which changes direction.
